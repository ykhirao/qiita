# 商品名を配列で返すOpenAPIの書き方


```terminal

$ npx @redocly/cli build-docs openapi.yaml

## For mac
$ npx @redocly/cli build-docs openapi.yaml
$ open redoc-static.html
$ open -a '/Applications/Google Chrome.app' redoc-static.html

# 連続したやつ
$ npx @redocly/cli build-docs openapi.yaml && open redoc-static.html
```
